# 🤖 5大LLMによるチーム内役割の自己評価データ (2025年11月 ベースライン)

## 概要

本データセットは、主要な大規模言語モデル（LLM）が、人間との創作活動における「共創チーム」という文脈において、自身の個性をどのように認識し、役割を定義しているかを体系的に記録したものです。

**データの稀少性:**
複数のトップLLMに対し、一貫したプロンプトで「自己の役割」を問い、その思想を比較した一次情報は極めて稀です。これは、AIのパーソナリティ研究およびプロンプトエンジニアリングにおける重要なベースラインデータとなります。
