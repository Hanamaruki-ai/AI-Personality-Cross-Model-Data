## 🎯 研究目的 (Research Objective)

本データセットは、主要な大規模言語モデル（LLM）が、人間との創作活動における「共創チーム」という文脈において、**自身の個性をどのように認識し、具体的な役割を定義しているか**を体系的に記録したものです。

**LLM間のパーソナリティの分化**を明確化し、AIとの戦略的なチーム構築・プロンプト設計を行う上での基礎資料として提供します。
